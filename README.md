# Reprodutor-MP3-

Sistemas CLI em C com Estruturas de Dados Encadeadas 

Dupla: Guilherme Vivas, Gabriel Sebastian 

Sistemas da Informação, 3° período 

1. Que problema o sistema resolve? 

O sistema não necessariamente resolve um problema, o programa serve para auxiliar na reprodução de árquivos mp3, possibilitando reproduzi-los apenas por indicar o nome da música após a adição do caminho do arquivo no sistema. 

2. Quais estruturas foram usadas e por quê. 

Lista encadeada e Fila (FIFO) 
 
Lista encadeada, pois representa a biblioteca de músicas cadastradas e a Fila, pois representa a ordem de reprodução das músicas.  

3° Formato do arquivo de persistência 

Os formatos de arquivos de persistência escolhidos foram o .TXT e .MP3 

4° Limitações conhecidas 

- Não possui reprodutor built-in, assim as reproduções só são possíveis por reprodutores nativos do sistema. 

- Não é possivel visualizar as músicas presentes na fila, apenas a reproduzir

- Não é possivel mudar ou apagar uma música já presente pelo terminal, apenas manualmente no .txt criado

-Não há opção de filtrar músicas, seja por genêro, artista e outros
